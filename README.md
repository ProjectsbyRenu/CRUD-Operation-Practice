# CRUD-Operation-Practice-
This is a simple CRUD practice project built to understand how data flows between the frontend, backend, and database. In this project, users can enter data through a form, and the application performs Create, Read, Update, and Delete (CRUD) operations using a database.
The main goal of this project is to practice backend development, database integration, and dynamic data rendering on the frontend.

Features:
Add new data through a user form
Store user data in the database
Display stored data on the frontend
Edit existing data
Delete records from the database
Basic UI to view and manage records

Technologies Used :
Node.js
Express.js
MongoDB
EJS / HTML / CSS / JavaScript

Purpose :
This project was created for practice and learning CRUD operations in a full-stack environment. It helps understand how to:
Handle user input
Connect backend with a database
Perform database operations
Render dynamic data on the frontend
